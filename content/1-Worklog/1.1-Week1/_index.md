### Week 1 Achievements:

* **Mastered AWS Identity and Access Management (IAM) Fundamentals:** Developed a solid understanding of IAM Users, Groups, Roles, and their roles in securing AWS resources.
* **Implemented Group-Based Access Management:** Created IAM Groups and assigned IAM Users to streamline permission management and enforce consistent access policies.
* **Configured Secure User Identities:** Provisioned IAM Users with appropriate permissions, following AWS security best practices for identity management.
* **Established Role-Based Access Control (RBAC):** Created IAM Roles with trust policies, enabling users and AWS services to securely assume temporary permissions without sharing long-term credentials.
* **Applied the Principle of Least Privilege:** Designed and assigned permissions that granted only the minimum access required for each user or role, reducing security risks.
* **Strengthened AWS Access Security:** Gained practical experience in separating identities from permissions using IAM Groups and Roles, improving scalability, maintainability, and overall security of AWS environments.
